# Stage

## Prerequisites

The following are VM requirements for a Pilot. As seen in the [Solution Architecture](/Prepare/learn#solution-architecture), the Windows VM will host the ADDI project and database,
while the Linux VM will host the Z Refactoring Assistant Server.

### Understand - ADDI Project & Database Host

#### **Hardware**
- CPU:  Intel Core i5 or equivalent
- RAM:  32 GB
- Disk: 35 GB

#### **Platform**
- OS:
    - Windows Server 2019 (Recommended)
    - Other Options:
        - Windows 10/11 Enterprise (64-bit)
        - Windows 10/11 Pro (64-bit)
        - Windows Server 2019/2022 Datacenter Edition (64-bit)
        - Windows Server 2019/2022 Standard Edition (64-bit)
- Database:
    - [Microsoft SQL Server 2019](https://www.microsoft.com/en-us/sql-server/sql-server-2019) (Recommended)
    - Other Options:
        - Microsoft SQL Server 2014/2016/2017/2022
        - Db2 Standard Edition 11.5.5.x (see: [fix pack](https://www.ibm.com/support/pages/node/6438025))

#### **Software**
- Eclipse OR IDz:
    - Eclipse:
        - 64 bit
        - [v4.23.x](https://www.ibm.com/links?url=https%3A%2F%2Fwww.eclipse.org%2Fdownloads%2Fpackages%2Frelease%2F2022-03%2Fr%2Feclipse-ide-rcp-and-rap-developers) (Recommended, latest)
            - or [v4.6.x](https://www.ibm.com/links?url=https%3A%2F%2Fwww.eclipse.org%2Fdownloads%2Fpackages%2Frelease%2Fphoton%2Fm5%2Feclipse-rcp-and-rap-developers) and above
            :::note
            versions older than 4.15 are only compatible with Java 8; v4.15 and v4.16 are compatible with both Java 8 and 11; newer versions are only compatible with Java 11 
            :::
    - [IBM Developer for Z Systems](https://www.ibm.com/links?url=https%3A%2F%2Fibm.github.io%2Fmainframe-downloads%2Fdownloads.html):
        - v16.x.x (Recommended)
            - or IDz 14/15 with Java 8 (Unsupported)
- [64 bit Oracle Java 11](https://www.oracle.com/java/technologies/downloads/#java11-windows) (Recommended)
    - or 64 bit [Oracle Java 8](https://www.oracle.com/java/technologies/downloads/#java8-windows)
- Supported Internet browsers: (used for accessing IBM AD Configuration Server)
    - Google Chrome (Recommended)
    - Mozilla Firefox
    - Microsoft Edge (verion 79 or later)

### Refactor - Z Refactoring Assistant Server

#### **Hardware**
- CPU:  4 cores
- RAM:  16 GB
- Disk: 500 GB

#### **Platform**
- OS: Red Hat Enterprise Linux (RHEL) version 8 or later on an AMD64 system

#### **Software**
- Required tools and packages:
    - podman version 4 or later
    - _podman-plugins_ package
    - _openssl_ command
    - _keytool_ command (from Java 8 or higher Runtime Environment)
    - VNC session to access via VNC client or SSH
- Network access to IBM Application Discovery and Delivery Intelligence (IBM ADDI)
    - Version 6.1.3 or later
- Supported Internet browsers: (used for accessing Z Refactoring Assistant)
    - Google Chrome (recommended)
    - Mozilla Firefox
    - Microsoft Edge (version 79 or later)


### Transform

For Transforming Cobol to Java, the only new hardware required is a Db2 instance on the cloud. Additional required software will live on the Windows machine.

#### **Platform**
- Db2 (on IBM Cloud)

#### **Software**
- IBM Service client 11.5 (64 bit)
- Data Generator
    - WCA4Z-Data-Generator zip file copied from the Refactoring Assistant to the Windows VM
- [Visual Studio Code](https://code.visualstudio.com/)
