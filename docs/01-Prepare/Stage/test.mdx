# Stage

## Pre-requisites

The following are VM requirements for a Pilot. As seen in the [Solution Architecture](/Prepare/learn), the Windows VM will host the ADDI project and database,
while the Linux VM will host the Z Refactoring Assistant Server. Multiple listings indicate valid options.

### ADDI Project & Database Host

#### **Hardware**
- CPU:  Intel Core i5 or equivalent
- RAM:  32 GB
- Disk: 35 GB

#### **Platform**
- OS:
    - Windows 10/11 Enterprise (64-bit)
    - Windows 10/11 Pro (64-bit)
    - Windows Server 2019/2022 Datacenter Edition (64-bit)
    - Windows Server 2019/2022 Standard Edition (64-bit)
- Database:
    - Db2 Standard Edition 11.5.5.x (see: [fix pack](https://www.ibm.com/support/pages/node/6438025))
    - Microsoft SQL Server 2014/2016/2017/2019/2022

#### **Software**
- Eclipse: 
    - 32/64 bit
    - [v4.6.x](https://www.ibm.com/links?url=https%3A%2F%2Fwww.eclipse.org%2Fdownloads%2Fpackages%2Frelease%2Fphoton%2Fm5%2Feclipse-rcp-and-rap-developers) \<\= version \>\= [v4.23.x](https://www.ibm.com/links?url=https%3A%2F%2Fwww.eclipse.org%2Fdownloads%2Fpackages%2Frelease%2F2022-03%2Fr%2Feclipse-ide-rcp-and-rap-developers) 
    - compatible Java version
- [IBM Developer for Z Systems](https://www.ibm.com/links?url=https%3A%2F%2Fibm.github.io%2Fmainframe-downloads%2Fdownloads.html):
    - v15.0.x \<\= version \>\= v16.x.x 
        - IDz 14 or 15 with Java 1.8 that is installed with IDz.
        - IDz 16 with Java 11 that is installed with IDz.
- Supported Internet browsers: (used for accessing IBM AD Configuration Server)
    - Google Chrome (recommended)
    - Mozilla Firefox
    - Microsoft Edge (verion 79 or later)

### Z Refactoring Assistant Server

#### **Hardware**
- CPU:  4 cores
- RAM:  16 GB
- Disk: 500 GB

#### **Platform**
- OS: Red Hat Enterprise Linux version 8 or later on an AMD64 system

#### **Software**
- Required tools and packages:
    - podman version 4 or later
    - _podman-plugins_ package
    - _openssl_ command
    - _keytool_ command (from Java 8 or higher Runtime Environment)
- Network access to IBM Application Discovery and Delivery Intelligence (IBM ADDI)
    - Version 6.1.3 or later
- Supported Internet browsers: (used for accessing Z Refactoring Assistant)
    - Google Chrome (recommended)
    - Mozilla Firefox
    - Microsoft Edge
